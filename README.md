# Portfolio

React-based developer portfolio showcasing Laravel backend projects and production systems.

## Key Features

- **Dynamic Contact Form**: Integrated with Google Sheets for storage and instant Email notifications.
- **Premium Design**: Modern aesthetic with glassmorphism, orbs, and clean typography.
- **Responsive Layout**: Optimized for all devices.

## Tech Stack

- Frontend: React, Vite, CSS (Vanilla)
- Backend Integration: Google Apps Script, Google Sheets API
- Deployment ready for Vercel/Netlify.
